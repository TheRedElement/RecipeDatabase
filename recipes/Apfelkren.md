# Apfelkren
![300](../gfx/PXL_20250420_025526263.jpg)

|Ingredient|Amount (4 portions)|
| :- | :- |
| apple | 2 |
| horseradish | - |
| vinegar | - |

## Recipe
1. grate **horseradish**
2. grate **apples**
3. mix
4. add a little **vinegar**

## Notes
* taste generally improves with more sweet and juicy **apples**
